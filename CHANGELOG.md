# Changelog

All notable changes to Webhook Inspector will be documented in this file.

## [0.1.0] - 2024-02-23

### Added

- Webhook capture and inspection with full request/response details
- JSON body viewer with collapsible tree structure
- One-click request replay
- Stripe CLI, GitHub, and Shopify webhook integration examples
- Status bar indicator with live proxy status
- In-panel settings (listen port, forward URL)
- English and Turkish language support with automatic locale detection
- Responsive layout with compact mode for narrow panels
- Pro features: Copy as cURL, HAR export
- LemonSqueezy license validation with 7-day caching and offline support
- Request filtering by status code (2xx, 4xx, 5xx, Error)
- Path-based search filtering
